# Lhůtovníček PWA v2.4

PIN: 158158

Novinky v2.4:
- automatické rozdělení do sektorů TČ, PŘ a ČJ podle čísla jednacího,
- TČ má základní lhůtu 2 měsíce,
- PŘ a ČJ mají základní lhůtu 30 dní,
- přehledové karty pro počty TČ/PŘ/ČJ,
- export CSV obsahuje i typ spisu.

Aplikace neobsahuje service worker/cache, aby se na iPhonu nezasekávaly staré verze.
